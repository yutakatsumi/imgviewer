# imgviewer
The instructions for how to configure this viewer (via HTML scripting) can be found on Papaya's GitHub page (https://github.com/rii-mango/Papaya).

## Building a page
```
./papaya-builder.sh -singlefile -images ../MNI152_T1_1mm_brain.nii.gz ../preSUV_n36.nii ../postSUV_n36.nii ../spmT_preSUV_vs_postSUV_masked.nii -local
```
